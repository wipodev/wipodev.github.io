## ğŸ§  Mechanics & Commands

### Main Commands

| Command                            | Description                                                     |
| ---------------------------------- | --------------------------------------------------------------- |
| `/playerclone <username>`          | Creates a single clone of the specified player.                 |
| `/playerclone <username> <amount>` | Creates multiple clones of that player.                         |
| `/playerclone <username> remove`   | Removes all clones with the given username.                     |
| `/playerclone ffa`                 | Starts a **Free-For-All** battle among all clones.              |
| `/playerclone team`                | Starts a **Team Battle** where each username fights as a group. |
| `/playerclone stop`                | Stops the current battle.                                       |
| `/playerclone help`                | Displays all available commands.                                |
